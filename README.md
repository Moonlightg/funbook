## Install

```bash
yarn add funbook
```

## Usage

### With Webpack

```js
import FunBook from 'funbook'

// Create a book
const book = new FunBook({
  title: 'My Awesome Book'
})

// Add a new chapter
book.chapter({
  title: 'Chapter 1',
  content: 'This is the first chapter of my book!',
  stories: [{
    title: 'Story 1',
    content: 'A chapter could contain multiple stories'
  }]
})

// Open the book at #root element
book.open('#root')
```
